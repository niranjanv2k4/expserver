# eXpServer

**eXpServer** is an educational web server built in C, designed to help students understand the inner workings of real-world web servers. It is part of a guided learning initiative to teach concepts like socket programming, HTTP handling, concurrency, TLS, reverse proxying, caching, rate limiting, and more — all from scratch.

🔗 **Project Site**: [https://expserver.github.io](https://expserver.github.io)

## Overview

This project is intended primarily for undergraduate students and educators in computer science, especially those studying computer networks or systems programming. Through a structured, incremental approach, learners implement a fully functional web server using Linux system calls and C APIs, gaining hands-on experience with:

- TCP/IP socket programming  
- HTTP protocol implementation  
- Serving static files  
- Multi-process/multi-threaded server design  
- Gzip compression  
- Reverse proxy and load balancing  
- TLS encryption  
- Caching and rate limiting
